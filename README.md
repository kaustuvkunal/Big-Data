### Installation  
 

 
 ##### Pre-requisite
 ```
   - Maven
   - Java 8
   - Hadoop 2.7.5 
   - IDE
   ```
##### Clone 

 `git clone git clone https://github.com/kaustuvkunal/Big-Data.git`

 ##### Build
 ```
 cd big-data
 mvn clean install
 ```
